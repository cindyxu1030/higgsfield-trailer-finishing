# higgsfield-trailer-finishing

An [Agent Skill](https://agentskills.io) that finishes Higgsfield-generated clips into one delivered video: **stitch → optional cinematic grade → original soundtrack matched to the assembled cut → mux.**

Generation is Higgsfield's job (see their official [`higgsfield-generate`](https://github.com/higgsfield-ai/skills) skill). This skill picks up where generation ends: your agent runs validated ffmpeg recipes to assemble and grade the cut, then Sonilo video-to-music composes an original track from the finished video itself — so the music follows the edits, and the length matches automatically. Every Sonilo track is licensed and safe for commercial use (terms apply).

**▶ [Example result](https://github.com/cindyxu1030/sonilo-video-to-music-cookbook/blob/main/assets/demo-trailer.mp4)** — clips generated on Higgsfield (Seedance 2.0 + Kling 3.0 Turbo), finished with this flow.

> Community skill built by the Sonilo team. Not an official Higgsfield product. Works with clips from any source; Higgsfield is the generator it chains with best.

## Install

Copy the skill into your agent's skills directory, e.g. for Claude Code:

```bash
git clone https://github.com/cindyxu1030/higgsfield-trailer-finishing ~/.claude/skills/higgsfield-trailer-finishing
```

## Prerequisites

- `ffmpeg` / `ffprobe` on `$PATH`
- For the soundtrack stage: your own `SONILO_API_KEY` from [sonilo.com](https://sonilo.com), registered via the [Sonilo MCP server](https://github.com/sonilo-ai/sonilo-mcp) (`claude mcp add sonilo --env SONILO_API_KEY=sk-... -- uvx sonilo-mcp`). Stitch, grade, and mux work without a key.
- The `higgsfield` CLI only if clips still need generating.
